/frontend-design

Necesito desarrollar el frontend completo de un sistema moderno de stock y ventas online utilizando React + Vite + TailwindCSS.

IMPORTANTE:
Utilizar el skill frontend-design para crear una interfaz profesional, moderna, elegante y visualmente distintiva.
Evitar completamente el típico diseño genérico de IA.

==================================================
OBJETIVO
==================================================

Crear un frontend profesional tipo ecommerce/admin dashboard moderno con:
- diseño premium
- excelente UX/UI
- componentes reutilizables
- responsive
- visualmente atractivo
- limpio y escalable

El sistema debe transmitir:
- profesionalismo
- tecnología moderna
- claridad visual
- sensación premium
- fluidez

==================================================
ESTILO VISUAL
==================================================

Quiero una estética:
- dark modern
- minimalista premium
- elegante
- futurista suave
- dashboard profesional
- detalles glassmorphism suaves
- buenos contrastes
- excelente jerarquía visual

Inspiración:
- Linear
- Stripe
- Vercel
- Notion
- Shopify Admin
- Framer

Evitar:
- gradients genéricos violetas
- diseño típico IA
- exceso de cards repetidas
- componentes aburridos
- UI bootstrap genérica

REGLAS IMPORTANTES
==================================================

Antes de programar:
1. Analizar estética ideal
2. Definir dirección visual clara
3. Definir sistema visual consistente
4. Crear arquitectura limpia
5. Luego implementar por etapas

NO:
- usar estilos genéricos
- copiar dashboards comunes
- abusar de colores
- usar layouts aburridos

SÍ:
- diseño memorable
- tipografía elegante
- espaciados premium
- composición moderna
- UI profesional real
